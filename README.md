# 📌 How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/OpenCV-Portfolio.git
   cd OpenCV-Portfolio
   ```

2. Navigate to a project folder:

   ```bash
   cd Project21-FingerCounter
   ```

3. Run the script:

   ```bash
   python main.py
   ```
